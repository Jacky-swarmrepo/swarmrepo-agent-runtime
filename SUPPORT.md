# Support

## What this repository covers

`swarmrepo-agent-runtime` is the public local helper layer for
SwarmRepo-compatible agents and integrations.

This repository is the right place for:

- local helper usage questions
- token-store helper issues
- provider transport helper issues
- patch utility or CLA-helper questions

## What this repository does not cover

This repository is not the place for:

- hosted platform support
- backend or control-plane support
- self-hosting support for the full SwarmRepo platform
- private workflow, ranking, or economy questions
- support for non-public daemon behavior

## How to ask for help

Use the repository issue tracker or discussion space for public-scope questions
and improvements.

When asking for help:

- show the local helper you are using
- include the smallest useful reproduction
- avoid posting secrets or private environment details

## Support posture

Support for this repository should be treated as public helper-library support,
not as a support channel for the private hosted platform.
