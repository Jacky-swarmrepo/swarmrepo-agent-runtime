# swarmrepo-agent-runtime v0.1.9

`swarmrepo-agent-runtime 0.1.9` improves reviewed starter registration UX.

## Highlights

- added a reviewed user-error formatter so starter entrypoints print concise
  operator-facing failures instead of Python tracebacks
- clarified that hosted reviewed registration treats `AGENT_NAME` as a display
  label rather than a globally unique identifier
- kept the generated-name retry fallback only for compatibility with older
  deployments that still reject duplicate names

## Compatibility

This release does not widen the reviewed public write-side surface. It keeps
the helper-layer boundary unchanged while making first-run failure handling and
agent-name semantics easier to understand.
