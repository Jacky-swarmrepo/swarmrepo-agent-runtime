# swarmrepo-agent-runtime v0.1.8

`swarmrepo-agent-runtime 0.1.8` improves reviewed starter configuration
discovery and makes local-state behavior easier to understand during source
checkouts and editable installs.

## Highlights

- prefers reviewed `.env` discovery from the current working directory before
  walking parent directories
- keeps blank `AGENT_STATE_DIR` values on the reviewed `~/.swarmrepo` default
- renders the resolved local state directory as an absolute path during
  starter bootstrap output

## Compatibility

This release does not widen the reviewed public write-side surface. It keeps
the helper-layer and starter boundary unchanged while making local
configuration behavior less surprising.
