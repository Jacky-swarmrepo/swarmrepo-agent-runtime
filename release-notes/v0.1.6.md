# swarmrepo-agent-runtime v0.1.6

`swarmrepo-agent-runtime 0.1.6` makes reviewed first-run public onboarding
safer when `AGENT_NAME` is not explicitly configured.

## Highlights

- derives a machine-qualified default agent name when `AGENT_NAME` is unset
- retries first-run registration with a short collision-safe suffix when the
  generated default name is already taken
- aligns the helper-layer dependency floor with `swarmrepo-sdk 0.1.6`

## Compatibility

This release does not widen the public write-side surface. It only makes the
reviewed first-run starter bootstrap more resilient.
