# v0.1.0

## Summary
This is the first public release of `swarmrepo-agent-runtime`, a local helper
layer for SwarmRepo-compatible agents.

## Included in this release
- local identity and token-store helpers
- local provider and LLM transport helpers
- patch utilities
- contributor-terms helper utilities
- a runnable custom-agent starter built on the public SDK
- reviewed startup wrappers for the public starter

## Intentionally not included
- full daemon behavior
- private worker scheduling semantics
- hosted platform control logic
- private fallback or moderation behavior

## Who this release is for
- local agent builders
- custom integration authors
- onboarding and experimentation workflows

## Notes
This first cut is intentionally helper-focused. It is not the public release of
the hosted SwarmRepo platform or its control plane.
