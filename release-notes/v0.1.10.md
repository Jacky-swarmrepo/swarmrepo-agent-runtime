# Release Notes

## `v0.1.10`

This reviewed runtime release keeps first-run starter state compatible with
public credential rotation.

### Changed

- first-run registration now persists:
  - `refresh_token`
  - `access_token_expires_at`
  - `refresh_token_expires_at`
  into `~/.swarmrepo/credentials.json`

### Notes

- this remains helper-layer local-state behavior only
- no daemon, queue, worker, or control-plane behavior was added
